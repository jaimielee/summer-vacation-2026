# 우리 가족 여름 휴가 2026 🏖️

이사·방학·휴가 일정을 **모바일에서 달력으로 한눈에** 보고, 날짜를 누르면 그날의
**시간대별 상세 계획**을 볼 수 있는 페이지입니다.

## 주요 일정

| 날짜 | 일정 |
|------|------|
| 7/27 | 📦 이사 |
| 7/28 ~ 7/31 | 재연 방학 |
| 7/29 ~ 7/31 | 지영 휴가 1차 |
| 7/30 ~ 8/7 | 하연 방학 · 정민 휴가+부분출근 |
| 8/6 ~ 8/7 | 지영 휴가 2차 |

## 수정하는 법

모두 순수 HTML/CSS/JS 라 빌드 도구가 필요 없습니다.

- **일정(달력 바) 바꾸기** → [`data.js`](data.js) 의 `EVENTS`
- **날짜별 상세 시간표 바꾸기** → [`data.js`](data.js) 의 `SCHEDULES`
- **표시할 월 바꾸기** → [`data.js`](data.js) 의 `MONTHS`
- **디자인/색상** → [`styles.css`](styles.css) 상단 `:root` 변수

고친 뒤 커밋 & 푸시하면 GitHub Pages에 자동 반영됩니다.

```bash
git add -A && git commit -m "일정 수정" && git push
```
